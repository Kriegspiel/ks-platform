# Assets

Design assets for the Kriegspiel platform — logos, social images, and brand materials.

## Contents

| Path | Purpose | Spec |
|---|---|---|
| `logo/` | Logo SVG files, favicon, apple-touch-icon | [LOGO.md](../LOGO.md) |
| `og-images/` | Open Graph social sharing images (1200×630) | [LOGO.md](../LOGO.md) |

## Expected Deliverables

### `logo/`

| File | Format | Description |
|---|---|---|
| `logo.svg` | SVG | Primary logo — double-headed knight, fill controlled by CSS variable |
| `favicon.svg` | SVG | Simplified logo optimized for 16×16 |
| `favicon.ico` | ICO | Multi-resolution (16, 32, 48px) |
| `apple-touch-icon.png` | PNG 180×180 | iOS home screen icon |

### `og-images/`

| File | Format | Description |
|---|---|---|
| `og-image-dark.png` | PNG 1200×630 | White knight on dark background + "Kriegspiel" text |
| `og-image-light.png` | PNG 1200×630 | Black knight on light background + "Kriegspiel" text |

## Generation Notes

These assets should be generated by an agent following the specifications in [LOGO.md](../LOGO.md) and [DESIGN.md](../DESIGN.md). The logo SVG must:

- Be a flat silhouette (no gradients, no fine detail)
- Use `currentColor` or a `fill` attribute that CSS custom properties can override
- Be optimized with SVGO (no embedded raster images, no embedded fonts)
- Work at all sizes from 16×16 to 512×512
